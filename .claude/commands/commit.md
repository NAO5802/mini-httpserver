# コミット実行

現在のブランチに対して、CLAUDE.md のコミットメッセージ規約に従ってコミットを作成する。

## 実行手順

### 1. 差分確認（並列実行）

```bash
git status
git diff
git diff --cached
git log --oneline -5
```

### 2. 変更内容の分析

- 差分から変更の種類を判断
- コミットメッセージを検討
- 変更がない場合は「コミット対象の変更がありません」と報告して終了

### 3. ステージング

```bash
git add -A
```

### 4. コミット作成

CLAUDE.md の規約に従ってコミットを作成する。

### 5. 結果確認

```bash
git status
git log --oneline -1
```

## 注意事項

- プッシュは行わない（ローカルコミットのみ）
